### 🚀 快速开始

#### 1. 本地开发
```bash
# 直接在浏览器中打开 index.html
# 或使用本地服务器
python3 -m http.server 8000
# 访问: http://localhost:8000
```

#### 2. 个性化配置

**修改个人信息：**
- 编辑 `index.html` 中的联系信息
- 替换 `images/avatar.jpg` 为你的头像

**修改作品内容：**
```html
<!-- 在 PAGE 03 作品案例中修改 -->
<div class="work-card">
    <div class="work-cover">
        <img src="images/work-1.jpg" alt="作品">
        ...
    </div>
    <div class="work-info">
        <h3>你的作品名称</h3>
        <p class="work-type">你的项目类型</p>
        ...
    </div>
</div>
```

**自定义颜色：**
```css
/* 在 style.css 中修改 CSS 变量 */
:root {
    --primary-color: #001a4d;      /* 深蓝 */
    --secondary-color: #6b2f9e;    /* 紫色 */
    --accent-color: #9d4edd;       /* 浅紫 */
    --dark-bg: #0d0221;            /* 背景黑 */
    ...
}
```
